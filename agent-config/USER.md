## 🎯 Agent 注意事项

### 向用户汇报时
- **必须**使用中文回复
- **必须**输出至少 200 字，确保信息完整
- **必须**包含：任务状态、完成摘要、验证结果、建议下一步
- **推荐**使用结构化格式（表格、列表、状态标记）

### 任务派发时
- 在任务描述中明确验收标准
- 设置合理的 `runTimeoutSeconds`
- 任务派发后**必须**调用 `sessions_yield()` 等待结果
- 在 `memory/YYYY-MM-DD.md` 中记录任务派发情况

### 错误处理
- 子任务失败时，分析错误类型（临时/代码/环境）
- 临时错误可以重试一次
- 无法自动恢复时，及时通知用户
- 多次超时后，建议用户分解任务
