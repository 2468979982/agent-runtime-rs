AGENT-RUNTIME-RS

一个高性能、模块化、可扩展的 Rust Agent 运行时框架，专为构建智能代理系统而设计。

⚡ 高性能

基于 Tokio 异步运行时，充分利用 Rust 的零成本抽象与内存安全特性，实现毫秒级响应与高吞吐处理能力。

🧩 模块化架构

核心 Runtime + 可插拔组件（LLM Adapter、Tool Executor、Memory Manager、Orchestrator），按需组合，灵活部署。

🌐 多协议支持

原生支持 OpenAI 兼容 API、Ollama、Anthropic、本地 GGUF 模型，无缝对接各类大模型后端。
